
# 📚 Estruturas de Dados em Python

Bem-vindo(a)! 👋

Este repositório reúne materiais introdutórios sobre as principais estruturas de dados nativas da linguagem Python. Cada conteúdo foi desenvolvido com explicações teóricas, exemplos práticos e exercícios para apoiar o aprendizado e a prática da programação.

## 📖 Materiais Disponíveis

### 📝 Listas (Lists)
As listas são coleções ordenadas e mutáveis, permitindo armazenar múltiplos valores em uma única variável. São amplamente utilizadas para manipulação de dados, iterações e armazenamento de informações que podem ser alteradas ao longo do programa.

🔗 Acesse: **[Lista](https://colab.research.google.com/drive/1jXnI7xPoMnblplbuq8YSg67sWGR9SfBj?usp=sharing)**

---

### 📌 Tuplas (Tuples)
As tuplas são coleções ordenadas e imutáveis. São indicadas para armazenar informações que não devem ser modificadas após sua criação, oferecendo maior segurança e desempenho em determinadas situações.

🔗 Acesse: **[Tupla](https://colab.research.google.com/drive/12egp_FNcoQjJeiRn3RrX_SJp7rCvBwAW?usp=sharing)**

---

### 🎯 Sets (Conjuntos)
Os sets são coleções não ordenadas que não permitem elementos duplicados. São muito úteis para eliminar repetições, realizar comparações e trabalhar com operações matemáticas de conjuntos, como união e interseção.

🔗 Acesse: **[Set](https://colab.research.google.com/drive/1gZfyd8P32nCJn66rGNHOnnG-z43x81Ja?usp=sharing)**

---

### 📚 Dicionários (Dictionaries)
Os dicionários armazenam informações no formato **chave: valor**, permitindo organizar dados de forma estruturada. São amplamente utilizados em sistemas, APIs, bancos de dados e aplicações que exigem acesso rápido às informações.

🔗 Acesse: **[Dicionário](https://colab.research.google.com/drive/11xPMz8UzCISBT6YzFH4nMqO2LLKiPnKL?usp=sharing)**

---
